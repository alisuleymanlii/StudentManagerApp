StudentManagerApp
=================

Creator: Ali Suleymanli

Description:
-------------
StudentManagerApp is a simple Java console application for managing student records.
You can add, delete, view students, calculate the average score, and find the highest score.
All student data is saved to a file (students.txt) for persistence.

Features:
----------
- Add new students with name, age, and score
- Delete students by their number
- View all students in a table format
- Calculate the average score of all students
- Find the student with the highest score
- Save and load student data from a file

Usage:
-------
1. Clone the repository:
   git clone https://github.com/YourUsername/StudentManagerApp.git

2. Compile the Java program:
   javac App.java

3. Run the program:
   java App

File Storage:
-------------
- The application uses a text file (students.txt) to store student records.
- Each line in the file represents a student: name,age,score.

Notes:
------
- Ensure you have Java installed.
- The console clears automatically when performing actions (may vary depending on OS).
